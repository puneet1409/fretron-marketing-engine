## AI Agents for Marketing (40-Agent Org)

From Jacob Bank (Relay.app): A marketing org with just the CEO + 40 AI agents doing the work of 5 people.

### The Six Core Functions

#### 1. Social Media Agents
- **Content research agents**: Monitor influencers (Dharmesh Shah, Jason Lemkin)
- **Content creation agents**: Produce posts across LinkedIn, X, YouTube
- **Engagement coach**: Analyzes monthly performance, recommends optimization
- **Reply agents**: Handle comments on CEO's posts

#### 2. Blog & Website Agents
- Blog content creation and promotion
- **Blog updater**: Monitors posts for freshness and SEO relevance
- **Competitor pricing tracker**: Scrapes websites, alerts on changes

#### 3. Email Agents
- Newsletter summarization (weekly content digests)
- **Lifecycle marketing agents**: Recommend next steps based on product data
- **Churn identification agents**: Reach out to at-risk customers

#### 4. Lead Qualification Agents
- Demo request classifier (via HubSpot forms)
- Classify leads: qualified, unqualified, or uncertain (for human review)
- CRM update automation

#### 5. Community & Events Agents
- Self-paced course personalization
- Welcome email and reminder sequences
- **Partner program reviewers**: Evaluate applications within 48 hours

#### 6. Partnerships Agents
- Partner engagement monitoring
- Recommendation engines (match customers with partners)

### Implementation Key
Successful AI agent adoption requires **"super delegators"** who:
- Map workflows
- Identify discrete tasks
- Create detailed instructions

**Critical Insight**: "AI agents struggle with some tasks we think are easy, but are WAY better at unexpected work"—like coaching executives or synthesizing complex reports.

### Fretron Applications
| Agent Type | Fretron Use Case |
|------------|------------------|
| Content research | Monitor logistics/TMS influencers |
| Lead qualification | Score inbound demo requests |
| Competitor tracker | Monitor SAP, Oracle, Locus pricing |
| Lifecycle email | Trigger based on product usage |
| Churn prevention | Alert on usage drops |

## Vibecoding for GTM

### What is Vibecoding?
Using AI coding tools (Lovable, Bolt, Replit, Cursor) to build functional web applications without coding knowledge.

**Key distinction from automation**: Not "if-this-then-that" workflows—actual web applications from scratch.

### The Tools

| Tool | Best For | URL |
|------|----------|-----|
| **Bolt** | React apps from prompts/spreadsheets, free Netlify hosting | bolt.new |
| **Lovable** | Interactive web products from natural language | lovable.dev |
| **Replit** | Rapid prototyping in cloud environment | replit.com |
| **Cursor** | AI-enhanced code editor (more technical) | cursor.com |

### GTM Use Cases

#### 1. ROI Calculators
Build interactive calculators that:
- Reframe value propositions for executives
- Capture prospect contact information
- Enable real-time calculation during sales conversations
- Generate financial projections automatically

**Example**: Replaced clunky Excel with visual tool in ~2 hours using Bolt.

#### 2. Lead Magnets
- Interactive assessment tools
- Benchmarking calculators
- Audit generators

#### 3. Landing Pages
- Custom landing pages without agency
- A/B test variants quickly
- Event-specific pages

#### 4. Product Demos
- Interactive prototypes
- Feature previews
- Concept validation tools

### Fretron Vibecoding Opportunities
1. **Freight Cost Calculator** - Interactive lane-by-lane analysis
2. **TMS Readiness Assessment** - Score their maturity
3. **Dispatch Efficiency Audit** - Upload data, get insights
4. **ROI Projector** - Calculate savings based on their volume

## Deep Research for GTM

### Core Principles

1. **Source Quality Control**
   - Direct AI toward high-quality sources
   - Request in-text citations
   - Include source credibility tables

2. **Contextual Richness**
   - Provide company background
   - Share operational details
   - State goals and constraints
   - Include prior work

3. **Research Planning**
   - Request upfront research plan before execution
   - Allows adjustments before spending time/credits

### The Prompt Framework

```
GOAL: [Ultimate objective + specific AI task]

CONTEXT:
- Company: [Details about Fretron, GTM motion, stage]
- Constraints: [Timeline, budget, resources]
- Prior work: [What you've already done]

CONTENT:
- Section 1: [What you need]
- Section 2: [What you need]
- Deliverables: [Specific outputs]

STYLE:
- Format: [Tables, pyramid principle, summaries]
- Length: [Word count if relevant]

SOURCES:
- Prioritize: [Official docs, industry reports]
- Avoid: [Outdated, low-quality]

INSTRUCTIONS:
- [Specific methodology notes]
- Ask me for missing context rather than assuming
```

### Five GTM Applications

#### 1. Large In-House Projects (ChatGPT)
- Marketing attribution systems
- Content strategy frameworks
- GTM planning processes

**Output**: 10,000+ word tactical guides with step-by-step implementation.

#### 2. Competitor Ad Analysis (GPT-5 Agent Mode)
- Analyze LinkedIn/Google ad libraries
- Extract messaging, positioning, targeting
- Include direct ad links and screenshots

#### 3. Homepage/Landing Page Audits (ChatGPT Agent Mode)
- Competitive analysis
- Improvement recommendations with mockups
- Ranked by impact vs. implementation difficulty

#### 4. Product Feature Comparison (ChatGPT)
- Synthesize competitor capabilities from docs
- Flag gaps and outdated claims
- Verification through supplementary searches

#### 5. Market Assessment for Expansion (GPT-5/Claude + ChatGPT)
- Two-step: Framework first, then research
- Ranked assessments using primary data

### Tool Selection Guide

| Tool | Best For |
|------|----------|
| **ChatGPT** | General-purpose; deepest, most rigorous reports |
| **Gemini** | Shares research plan automatically; generous limits |
| **Perplexity** | Website-specific focus; superior source control |
| **Claude** | 1,000-2,000 word sleek reports; excellent entry points |

### Fretron Deep Research Applications
1. **Competitor Analysis**: SAP TM, Oracle, Locus feature comparison
2. **Industry Reports**: Freight cost benchmarks by sector
3. **Market Sizing**: Addressable market by industry vertical
4. **Content Research**: Topic ideation with source validation

## ABM to ABX Evolution

### The Shift
**ABM** (Account-Based Marketing) → **ABX** (Account-Based Experience)

| ABM (Old) | ABX (New) |
|-----------|-----------|
| Marketing-owned | Unified sales, marketing, product |
| Single channel | Multi-signal orchestration |
| Siloed data | Shared signals and objectives |
| Campaign-focused | Experience-focused |

### Multi-Signal Approach

Modern ABX leverages:
- **Product usage data**: How users interact with product
- **Intent signals**: Webinar attendance, email engagement, website activity
- **Firmographic data**: Company size, industry, geography
- **Technographic data**: Software companies use
- **Growth signals**: Job postings, funding, headcount changes

**Key Insight**: "Product usage data isn't the only important signal" even for PLG companies.

### AI in ABX

**Where AI works best**: Top and middle of funnel
- Account identification using historical data
- Automated research for smarter outbound
- Personalized outreach at scale

**Reserve humans for**: Later-stage deal work, complex negotiations

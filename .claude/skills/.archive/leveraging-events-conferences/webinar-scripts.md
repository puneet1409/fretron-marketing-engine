## Webinar Types

### Type 1: Educational Webinar (Lead Gen)

**Purpose**: Generate new leads through valuable content
**Target**: ToFU/MoFU prospects
**Duration**: 45-60 minutes
**Frequency**: Monthly

**Topics**:
- "How to Reduce Freight Costs 15-25% in Manufacturing"
- "The Real Cost of Excel-Based Logistics Management"
- "5 KPIs Every Logistics Leader Should Track"

### Type 2: Customer Story Webinar (BoFU)

**Purpose**: Convert pipeline with proof
**Target**: MoFU/BoFU prospects
**Duration**: 30-45 minutes
**Frequency**: Quarterly

**Topics**:
- "How [Customer] Cut Tracking Calls by 85%"
- "From Excel Chaos to Real-Time Visibility: A Customer Story"
- "â‚¹1.2 Cr Saved: Inside [Customer]'s Logistics Transformation"

### Type 3: Product Demo Webinar (BoFU)

**Purpose**: Scale demos, accelerate pipeline
**Target**: BoFU prospects
**Duration**: 30-45 minutes
**Frequency**: Bi-weekly

**Topics**:
- "Fretron TMS: Live Product Tour"
- "See How Fretron Handles [Specific Use Case]"
- "Q&A with Fretron Product Team"

### Type 4: Industry Expert Panel (Thought Leadership)

**Purpose**: Build authority, capture new audience
**Target**: ToFU prospects
**Duration**: 60 minutes
**Frequency**: Quarterly

**Topics**:
- "The Future of Manufacturing Logistics in India"
- "Panel: How Logistics Leaders are Tackling Cost Pressures"
- "Steel/Cement/FMCG Logistics: What's Working in 2025"

## Webinar Tech Setup

### Platform Requirements

- Reliable streaming (Zoom, Teams, Webex)
- Registration page integration
- Recording capability
- Q&A/Chat functionality
- Polling (optional but useful)
- Screen sharing

### Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Audio echo | Use headphones, mute when not speaking |
| Slides not advancing | Use backup computer, share slides file |
| Speaker video freezing | Turn off video, use audio only |
| Low attendance | Send last-minute reminder, extend start |
| Too many questions | Address top questions, follow up by email |

## Engagement Best Practices

### Good Comment Examples

**Adding Value**:
```
Great point about [specific thing]. We've seen similar -
[add your perspective or data]. One thing I'd add is
[additional insight that builds on their point].
```

**Asking a Question**:
```
Really interesting perspective. Curious - how did you
approach [specific aspect]? We've been experimenting
with [related approach] and seeing [result].
```

**Sharing Experience**:
```
This resonates. When we [did similar thing], the biggest
lesson was [lesson]. Did you find the same?
```

### Bad Comment Examples (Avoid)

```
"Great post!"
"So true!"
"Thanks for sharing!"
"Love this!"
"Agreed!"
```

These add no value and don't build relationships.

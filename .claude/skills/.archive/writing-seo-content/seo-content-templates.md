## On-Page SEO Checklist

### Title Tag
- [ ] Primary keyword at beginning
- [ ] Under 60 characters
- [ ] Compelling (not just keyword stuffed)
- [ ] Year if relevant

### Meta Description
- [ ] Under 155 characters
- [ ] Includes primary keyword
- [ ] Has call-to-action
- [ ] Unique value proposition

### URL
- [ ] Short and descriptive
- [ ] Includes primary keyword
- [ ] Hyphens between words
- [ ] No special characters

### Headers
- [ ] H1 contains primary keyword
- [ ] H2s cover main subtopics
- [ ] H3s for supporting sections
- [ ] Keyword variations in headers

### Content
- [ ] Primary keyword in first 100 words
- [ ] Keyword density 1-2%
- [ ] Related keywords included naturally
- [ ] Answers search intent completely

### Internal Links
- [ ] Links to related pillar content
- [ ] Links to supporting cluster content
- [ ] Descriptive anchor text
- [ ] No broken links

### Images
- [ ] Alt text with keywords
- [ ] Compressed for speed
- [ ] Descriptive file names
- [ ] Relevant to content

### Schema
- [ ] Article schema
- [ ] FAQ schema if applicable
- [ ] Organization schema
- [ ] Breadcrumb schema

# Content Refresh Framework

Systematic approach to updating, optimizing, and maintaining existing content.

## Content Audit Framework

### Audit Dimensions

| Dimension | What to Assess | Tools/Methods |
|-----------|---------------|---------------|
| **Performance** | Traffic, conversions, engagement | Analytics |
| **Accuracy** | Facts, stats, information current | Manual review |
| **Relevance** | Still matches ICP needs | Content-ICP mapping |
| **SEO Health** | Rankings, technical issues | SEO tools |
| **Competitive** | How we compare to SERP | SERP analysis |
| **Brand** | Aligned with current messaging | Voice audit |

### Content Scoring Matrix

**Score each piece 1-5 on:**

| Criteria | 1 (Poor) | 3 (Moderate) | 5 (Excellent) |
|----------|----------|--------------|---------------|
| Traffic trend | Declining 20%+ | Flat | Growing 20%+ |
| Conversion rate | <0.5% | 0.5-2% | >2% |
| Search ranking | Page 3+ | Page 2 | Page 1 |
| Information accuracy | Multiple outdated items | Minor updates needed | Current |
| Brand alignment | Off-brand | Partially aligned | Fully aligned |

**Total Score Interpretation:**

| Score | Category | Action |
|-------|----------|--------|
| 20-25 | Top Performer | Light refresh, promote more |
| 15-19 | Solid | Standard refresh |
| 10-14 | Underperformer | Major refresh or consolidate |
| 5-9 | Poor | Unpublish or rewrite entirely |

## Refresh Priority Framework

### Priority Matrix

```
                    HIGH TRAFFIC
                         │
         REFRESH         │         OPTIMIZE
         URGENTLY        │         (Quick wins)
                         │
    ─────────────────────┼─────────────────────
                         │
         CONSIDER        │         REFRESH
         UNPUBLISH       │         STRATEGICALLY
                         │
                    LOW TRAFFIC

    DECLINING ←──────────┼──────────→ STABLE/GROWING
       PERFORMANCE              PERFORMANCE
```

### Refresh Categories

**Category 1: Quick Wins (1-2 hours)**
- Update statistics/dates
- Fix broken links
- Refresh CTAs
- Add new internal links
- Update meta descriptions

**Category 2: Content Enhancement (4-8 hours)**
- Add new sections
- Update examples/case studies
- Improve structure/readability
- Enhance visuals
- Add FAQ section

**Category 3: Major Refresh (1-2 days)**
- Rewrite significant portions
- New keyword targeting
- Complete restructure
- New visuals/media
- Expanded scope

**Category 4: Strategic Rewrite (3-5 days)**
- Complete content overhaul
- New angle/positioning
- Comprehensive research
- Professional design
- Promotion plan

## Content Consolidation Framework

### When to Consolidate

**Consolidation Signals**:
- Multiple posts on same topic
- Similar posts cannibalizing keywords
- Old posts with declining traffic
- Thin content that could be combined
- Duplicate or near-duplicate content

### Consolidation Process

```
1. IDENTIFY CANDIDATES
   - Same/similar target keyword
   - Same topic, different angles
   - Could be combined into comprehensive piece

2. ANALYZE PERFORMANCE
   - Which has best rankings?
   - Which has most backlinks?
   - Which has best engagement?

3. DECIDE APPROACH
   Option A: Keep best, redirect others
   Option B: Merge into new comprehensive piece
   Option C: Update best, delete others

4. EXECUTE
   - Create consolidated content
   - Set up 301 redirects
   - Update internal links
   - Monitor for issues

5. MONITOR
   - Track rankings post-consolidation
   - Watch for traffic changes
   - Check redirect functionality
```

## Quick Reference: Content Refresh Priorities

### Immediate Action (This Week)

1. **Broken links** - Fix all broken outbound links
2. **Outdated statistics** - Update any stats older than 2 years
3. **Wrong information** - Correct any factually incorrect content
4. **Dead CTAs** - Fix CTAs pointing to removed offers/pages

### High Priority (This Month)

1. **Top 10 traffic pages** - Quick refresh pass
2. **High-converting pages** - Optimization focus
3. **Declining pages** - Diagnose and address
4. **Competitive keywords** - SERP analysis and gaps

### Standard Priority (This Quarter)

1. **All blog content** - Audit and prioritize
2. **Case studies** - Update with new data
3. **Landing pages** - A/B test learnings
4. **Gated content** - Ensure still valuable

### Low Priority (This Year)

1. **Archive content** - Decide keep/consolidate/remove
2. **Thin content** - Expand or consolidate
3. **Low-traffic pages** - Strategic decision
4. **Format updates** - New visuals, video, interactive
